# 股票分析器 - 环境配置说明

## Python 依赖（启动器）

安装命令：
```bash
pip install -r requirements.txt
```

## Node.js 依赖（前端 + 后端）

### 首次安装
进入 `stock-analyzer` 目录后执行：
```bash
npm install
```

### 后端对话历史服务依赖
已包含在 npm install 中，使用的是：
- express（Web服务框架）
- cors（跨域请求支持）

## 启动说明

### 1. 启动后端服务（对话历史保存）
```bash
cd stock-analyzer
node chatHistoryServer.js
```
服务运行在 http://localhost:3001
对话历史保存在 `chat_history/` 目录（项目根目录下）

### 2. 启动前端
```bash
cd stock-analyzer
npm run dev
```
服务运行在 http://localhost:5173

### 3. 使用启动器（可选）
```bash
python 启动器.py
```
启动器会自动运行前端服务并打开浏览器
